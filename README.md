# Sistema de Roteiro Turístico

> Sistema para gerenciar destinos turísticos usando Árvore Binária de Busca

## Sobre

Este projeto foi desenvolvido para a disciplina de **Estruturas de Dados**.
O sistema permite gerenciar uma lista de destinos turísticos de forma eficiente
utilizando uma Árvore Binária de Busca (ABB).

##  Funcionalidades

- Adicionar destinos turísticos
- Remover destinos existentes
- Buscar destinos rapidamente
- Visualizar roteiro em ordem alfabética
- Analisar altura da árvore
